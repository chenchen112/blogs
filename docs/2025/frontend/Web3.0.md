# 初探 Web3.0

## 发展历程

### Web 1.0

平台创造，平台所有，平台控制，平台分配，平台收益

### Web 2.0

用户创造，平台所有，平台控制，平台分配，平台收益

该模式存在以下缺陷：
- 所有权和收益权不合理
- 平台垄断，控制用户
    - 算法推送
    - 广告
    - 电商杀熟
- 数据割裂，无法共享，一篇文章需要各个平台发一遍
- 隐私问题，聊天记录，购物记录，搜索记录
- 可信度问题，文章阅读量可能是假的，热门视频并不是热门
- 安全性问题，平台没了，数据都没了

### Web3.0

用户创造，用户所有，用户控制，用户分配，用户收益

- 用户授权个人数据
- 数据存储方案：去中心化的区块链技术可用以解决信任问题
